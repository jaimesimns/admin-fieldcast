ADMIN · SUPABASE SETUP
Connect to Supabase to store your markers, experiences, and media files in the cloud.
1
Create a free project at supabase.com
2
Go to Storage → New Bucket, name it fieldcast-media, set to Public
3
In SQL Editor, paste and run this schema:
create table markers ( id text primary key, title text not null, description text, type text not null, lat float8, lon float8, radius int4 default 30, media_url text, media_name text, direction text, scan_type text, created_at timestamptz default now() ); create table experiences ( id text primary key, title text not null, tagline text, description text, cover_url text, duration text, status text default 'draft', marker_ids text[] default '{}', created_at timestamptz default now(), updated_at timestamptz default now() ); alter table markers enable row level security; alter table experiences enable row level security; create policy "Public read" on markers for select using (true); create policy "Public read" on experiences for select using (true); create policy "Anon write" on markers for all using (true); create policy "Anon write" on experiences for all using (true);
4
Copy your Project URL and anon/public key from Settings → API
